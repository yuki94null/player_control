# アドバンスドコントロール

## 概要

マインクラフトにアクションを追加します(実装されているもの✅)

- ✅ウォールラン
- ✅ウォールジャンプ
- ✅ウォールキック
- ✅ハング
- ✅マントル
- ✅クロウル
- スライディング
- 座る

## 設定

`/trigger yrh.010.setting`を実行すると、設定画面を出すことができます

設定画面ではそれぞれのアクションの有効、無効を切り替えることができます

## 仕様

それぞれのアクションの仕様を解説します

### ウォールラン

ウォールランを発動するには以下の条件をクリアする必要があります

- 壁際にいる
- 壁が視点と同じ方向に延びている
- `Sprint`と`Jump`と`Forward`を入力中である
- 落下中である
- ウォールランができる時間が残っている
- ハング中でない

ウォールラン状態は1.5秒間維持され、
ウォールラン中に壁と反対側の移動キーを入力すると、ウォールジャンプが発動します

ウォールランを開始し、一度何らかの方法でウォールラン状態から脱しもう一度同じ方の壁でウォールランを行うことはできません

継続してウォールランし続けられるのは1.5秒、
ウォールジャンプなどで着地せずに、反対側の壁で連続してウォールランをすることができるのは6秒です
ただしウォールジャンプにも回数制限があるので注意する必要があります

ウォールラン中は、地面に足をつくまで移動スピードに倍率がかかりますが、ウォールランの移動は通常の移動とは異なるので、移動速度の影響はほとんどないと考えています
専ら視野角変化による演出用の側面が強いです

ウォールランは通常の階段などの利用を妨げないよう、落下が始まってからのみ利用できます

### ウォールジャンプ

ウォールジャンプを発動するには以下の条件をクリアする必要があります

- ウォールラン中である
- 壁と反対側の移動キーを入力する
- ウォールジャンプできる回数が残っている

ウォールラン状態の時に壁と反対側の移動キーを入力すると、ウォールジャンプが発動します
ex.壁が左側にある場合は`right`、壁が右側にある場合は`left`

ウォールジャンプの推進力は、角度が"進行方向"から"壁の反対側"方向にが45°になるように、毎発動時一定値で加算されます

ウォールジャンプが一連のウォールラン中にできる回数は3回になっています

### ウォールキック

ウォールキックを発動するには以下の条件をクリアする必要があります

- 壁際にいる
- 壁が後方にある
- `Sprint`と`Jump`と`BackWord`を入力中である
- 空中である
- ウォールラン中でない
- 空中に出てからまだウォールキックを発動していない

ウォールキックは、壁と直行する方向に離れるように推進力が働きます
これは視点の角度によらず直行する方向であることは変わりません

ウォールキックを発動するには単に空中であればよいので、
ジャンプによる上方向の慣性と組み合わせると上方向への高い跳躍が可能です

ウォールキックは一度発動すると着地するまでは利用できません

### ハング

ハングを発動するには以下の条件をクリアする必要があります

- 壁際にいる
- 壁が前方にある
- 前方の壁の上部に空間がある
- `Left`と`Right`を入力中である
- ハングができる時間が残っている
- マントル後でない

ハングを行うと、崖に捕まることができます
ハングには時間制限があり、着地せず連続で行えるのは３秒までです

マントル後にハングしなおすことはできません

### マントル

マントルを発動するには以下の条件をクリアする必要があります

- ハング中である
- `Jump`を入力している

マントルを行うと、捕まっている崖から飛び上がることができます

飛び上がった後上昇中に`Sprint`を入力していると、前方に加速します

マントル後にハングしなおすことはできません

飛び上がった後に`Sneak`状態になると、ブーストの判定を消すことができます（内部的にはブーストをしたことにします）

### クロウル

クロウルを発動するには以下の条件をクリアする必要があります

- 接地している
- `Sneak`と`Left`と`Right`を同時に入力している

### 座る

シットを発動するには以下の条件をクリアする必要があります

- `Sprint`を入力中である
- 接地している
- 十分に下を向く
- 上記の条件を0.75秒満たし続ける
